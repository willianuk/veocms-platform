# VeoCMS Platform

A Git-native CMS with a Notion-like editor for static blogs.

For developers using Next.js, Astro, or any Markdown-based blog.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Features

- **Notion-like Editor** - Beautiful visual editing powered by Novel
- **Git Workflow** - Drafts in branches, publish by merging to main
- **GitHub Native** - Connect your repo, edit posts, push changes
- **Framework Agnostic** - Works with Next.js, Astro, Gatsby, or any Markdown-based blog

## Quick Start

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)
